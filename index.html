audioContext: el AudioContext que estás usando. clipLevel: el nivel (0 a 1) que consideraría "recorte". El valor predeterminado es 0,98. promediado: cuán "suavizado" le gustaría que el medidor fuera con el tiempo. Debe estar entre 0 y menos de 1. El valor predeterminado es 0,95. clipLag: cuánto tiempo desea que se muestre el indicador de "recorte" después de que se haya producido el recorte, en milisegundos. El valor predeterminado es 750 ms. meter.checkClipping (); devuelve verdadero si el nodo se ha recortado en los últimos milisegundos de clipLag. meter.shutdown (); Se usa para destruir el nodo (es importante desconectar y eliminar el controlador de eventos para cualquier ScriptProcessor).
